# Search algorithm

> **查找定义：**根据给定的某个值，在查找表中确定一个其关键字等于给定值的数据元素

> 1）静态查找和动态查找；
>
> 　　　　注：静态或者动态都是针对查找表而言的。动态表指查找表中有删除和插入操作的表。
>
> 2）无序查找和有序查找。
>
> 　　　　无序查找：被查找数列有序无序均可；
>
> 　　　　有序查找：被查找数列必须为有序数列。

#### 顺序查找

> 就是按照字面意思：按照顺序一个一个的比较，顺序查找一般针对被查找序列是无序的，是一种暴力查找

```javascript
function orderFind (arr, data) {
  let len = arr.length;
  for (let i = 0; i <= len - 1; i++) {
    if (data === arr[i]) return i;
  }
  return -1;
}
```
